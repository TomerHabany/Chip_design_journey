The Chip Designer's Guide to the Galaxy

Chronicles of RTL Engineering and SystemVerilog Exploration

Project Overview & Mission

I am a recent engineering graduate documenting my transition from university theory to practical silicon engineering. Moving into the world of hardware description languages (HDLs) and professional workflows is a significant challenge, and I created this project to bridge that gap through structured, hands-on learning.

The Project Goal: My objective is to build a functional library of digital modules from the ground up, starting with basic arithmetic and progressing toward integrating a RISC-V CPU. I am documenting this process to demonstrate the architectural reasoning and functional verification strategies required for professional RTL design.

In this project, you can expect to find:

  • Tooling & Environment: A roadmap for setting up a professional-grade local environment using open-source tools like Verilator, GTKWave, and Makefiles.
  • RTL Design: Synthesizable SystemVerilog modules built with a focus on clean, scalable architecture.
  • Functional Verification: Self-checking testbenches and reference models used to ensure logic correctness before system integration.

For Fellow Graduates: I have designed these logs to be the guide I wish I had when I started. By following these entries, a fresh engineer can easily set up their own development environment, master the essential tools, and begin building their own hardware blocks. There is a lot to learn in this field, and I am focused on building a solid technical foundation one module at a time.
